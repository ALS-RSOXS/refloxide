# Roughness model: graded interface

## Scope

This page will document the graded-interface approach, where each rough
boundary is replaced by a finite set of thin sublayers that approximate a
continuous material profile.

## Use regime

- Large roughness
- Large index contrast
- Cases where simple multiplicative roughness factors lose validity

## Implementation notes

- Deterministic profile discretization strategy
- Control over sublayer count and profile family
- Convergence criteria based on reflected and transmitted observables

## Validation

- Convergence under sublayer refinement
- Reduction to sharp-interface behavior in the zero-width limit
